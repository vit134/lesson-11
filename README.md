# ШРИ-2018. Домашнее задание Bem

## Описание

Приложение построено на основе bem-react-boilerplate.

Уровни переопределения:
  - Common
  - Tablet
  - Phone

С помощью пакета device реализовано определение девайся пользователя на сервере.

## Запуск

```
npm i
npm run build
npm run server
http://localhost:3001
```
